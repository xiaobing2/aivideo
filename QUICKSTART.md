# 快速启动指南

## 第一步：安装依赖

```bash
npm install
```

## 第二步：启动应用

```bash
npm run dev
```

浏览器会自动打开 `http://localhost:3000`

## 第三步：配置API密钥

首次启动时，应用会自动弹出设置对话框，或者点击右上角的"⚙️ 设置"按钮：

1. 访问 [阿里云DashScope控制台](https://dashscope.console.aliyun.com/)
2. 注册/登录账号
3. 进入"API-KEY管理"页面
4. 创建新的API密钥
5. 将密钥粘贴到设置对话框中
6. 点击"保存"

**注意**：
- API密钥保存在浏览器本地存储中，不会上传到服务器
- 如果没有配置API密钥，生成评论时会提示配置

## 使用流程

1. **配置API密钥**（首次使用）：
   - 启动应用后会自动弹出设置对话框
   - 或点击右上角"⚙️ 设置"按钮
   - 输入并保存API密钥

2. **允许权限**：首次使用时，浏览器会请求摄像头和麦克风权限，请点击"允许"

3. **输入话题**：
   - 在右侧输入框输入话题（例如："人工智能的未来发展"）
   - 点击"生成评论"按钮
   - AI会自动生成5条相关评论

4. **开始录制**：
   - 点击左侧"开始录制"按钮
   - 录制过程中可以暂停/继续
   - 点击"停止录制"后，视频会自动下载

5. **自动生成评论**（可选）：
   - 勾选"自动生成评论"选项
   - 仅在录制时生效
   - 可设置生成间隔（3秒/5秒/10秒/15秒）

## 常见问题

### 1. 无法访问摄像头/麦克风
- 检查浏览器权限设置
- 确保使用HTTPS或localhost
- 某些浏览器需要手动允许权限

### 2. API调用失败
- 点击右上角"⚙️ 设置"检查API密钥是否正确
- 确认API密钥未过期
- 检查网络连接
- 如果API不可用，应用会使用备用评论功能

### 3. 视频格式问题
- 默认使用WebM格式
- 如需MP4格式，需要修改 `VideoRecorder.tsx` 中的MediaRecorder配置
